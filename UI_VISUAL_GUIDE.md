## Animation Specifications

### Page Load
- Duration: 0.3s
- Effect: slideUp (opacity + translateY)
- Easing: ease-out

### Form Focus
- Duration: 0.3s
- Effect: Border color change + box-shadow
- Easing: ease

### Button Hover
- Duration: 0.3s
- Effect: translateY(-2px) + box-shadow
- Easing: ease

### Button Active
- Duration: immediate
- Effect: Return to normal
- Easing: N/A

### Error/Success Messages
- Duration: 0.3s
- Effect: slideDown (opacity + translateY)
- Easing: ease
